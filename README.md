# NSS Exercise: Colored-Reindeer

### Description:

> Build a javascript file to select colors for reindeer.

### How to run :
```
git clone https://github.com/Megraohoh/colored-reindeer

http-server -p 8080
```

This will show in your browser at:
`http://localhost:8080`
